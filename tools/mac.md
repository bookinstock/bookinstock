# Mac

## Basic

cmd + opt => alfred

cmd + space => spotlight

cmd + option + d => hide/show dock

cmd + h => hide current window

cmd + tab => switch application

## Files

opt + drag => copy file

## Screenshots

cmd + shift + 3

cmd + shift + 4

cmd + shift + ctrl + 4

cmd + shift + a

## Open File

cmd + shift + h => home

cmd + shift + d => desktop

cmd + shift + u => utilities
